DESCRIPTION :Task 1: File Handling Utility
In this task, I developed a Java-based File Handling Utility capable of performing essential file operations such as creating, reading, writing, appending, and modifying text files. The application was designed using Java's File I/O classes, including File, FileReader, FileWriter, BufferedReader, and BufferedWriter. Exception handling was implemented to ensure smooth execution and prevent crashes caused by invalid file paths or input errors. This project helped me understand how Java interacts with the file system and how data can be stored and retrieved efficiently. The application provides a simple and user-friendly way to manage text files while demonstrating the practical use of file handling concepts.
What I Learned:Java File I/O operations,Reading and writing text files,Exception handling,Resource management,Basic file system operations
Editor/Platform Used:Visual Studio Code (VS Code) with Java Extension Pack, JDK 17, and Command Prompt for compilation and execution.

Task 2: REST API Client
For this task, I created a Java application that consumes a public REST API to fetch real-time weather data and display it in a structured format. The application uses Java's HttpClient to send HTTP GET requests and processes JSON responses using a JSON parsing library. The retrieved data is displayed in an easy-to-read format, providing users with current weather information. Throughout this project, I gained practical experience in integrating external APIs into Java applications and handling network communication efficiently. Proper exception handling was also implemented to manage connectivity issues and invalid responses.
What I Learned:REST API concepts,HTTP requests using Java HttpClient,JSON parsing,API integration,Network communication,Error handling
Editor/Platform Used:Visual Studio Code (VS Code), Java Development Kit (JDK 17), OpenWeatherMap API, and GitHub for version control.

Task 3: Multithreaded Chat Application
In this task, I developed a client-server chat application using Java Socket Programming and Multithreading. The server accepts multiple client connections simultaneously by creating a separate thread for each connected client. Messages sent by one client are broadcast to all other connected clients in real time, enabling seamless communication. The project demonstrates the implementation of networking concepts, socket communication, and concurrent programming using Java threads. Proper connection management and exception handling ensure stable communication between the server and multiple clients. This application strengthened my understanding of real-time networking and client-server architecture.
What I Learned:Socket Programming,Client-server architecture,Multithreading,Real-time communication,Thread management,Network programming,Exception handling
Editor/Platform Used:Visual Studio Code (VS Code), Java JDK 17, Terminal/Command Prompt for execution, and GitHub for source code management

Task 4: AI-Based Recommendation System
In this task, I built a simple AI-based recommendation system using Java to suggest products based on user preferences and sample data. The application analyzes user interests and generates personalized recommendations using recommendation logic inspired by collaborative filtering techniques. This project introduced me to the fundamentals of recommendation systems and how artificial intelligence can enhance user experiences by providing relevant suggestions. I also learned how to process datasets, apply recommendation algorithms, and organize data efficiently. The project demonstrates the basic principles of AI-driven decision-making while improving my understanding of Java programming and data analysis.
What I Learned:Recommendation system concepts,User preference analysis,Basic AI and machine learning concepts,Data processing,Java collections,Algorithm implementation
Editor/Platform Used:Visual Studio Code (VS Code), Java JDK 17, GitHub for version control, and sample datasets for testing the recommendation engine.
